# 🗓️ Sistema de Agendamento de Tarefas

Um sistema simples e eficiente para gerenciamento e organização de tarefas diárias, permitindo o agendamento, controle e acompanhamento de atividades em um único lugar.

## 🚀 Funcionalidades

* 📌 Cadastro de tarefas
* ⏰ Agendamento por data e horário
* ✅ Marcar tarefas como concluídas
* ✏️ Edição e exclusão de tarefas
* 📋 Listagem organizada de atividades

## 🎯 Objetivo

Facilitar a organização pessoal e profissional, ajudando o usuário a manter controle total sobre suas tarefas e compromissos.

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* (Opcional) Python / Flask

## 💡 Possíveis aplicações

* Uso pessoal (organização diária)
* Pequenos negócios (agendamento de clientes)
* Profissionais autônomos

## 📈 Melhorias futuras

* Sistema de login
* Notificações de tarefas
* Integração com banco de dados
* Interface responsiva
* Dashboard de produtividade

## 📦 Como executar

Basta abrir o arquivo `index.html` no navegador.
